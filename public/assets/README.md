# Assets de Rutas y Sabores

- `videos/`: clips copiados desde las descargas de WhatsApp y renombrados como `rutas-video-01.mp4`, `rutas-video-02.mp4`, etc.
- `images/`: carpeta reservada para fotos propias del proyecto. Cuando estén los archivos originales, conviene guardarlos como `rutas-foto-01.jpg`, `rutas-foto-02.jpg`, etc. y reemplazar las rutas del carrusel en `../index.html`.
